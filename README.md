# 🌿 FloraVision

A modern, responsive plant e-commerce landing page built with **React**, **Vite**, **Tailwind CSS**, and **Framer Motion**. FloraVision showcases a premium UI with glassmorphism effects, smooth animations, reusable components, and a fully responsive layout.

---

## 🚀 Live Features

* 🌱 Modern plant store landing page
* 📱 Fully responsive design
* ✨ Smooth animations using Framer Motion
* 🎨 Glassmorphism UI
* 🛒 Product showcase section
* 🌿 Trending plants section
* ⭐ Customer review section
* 🍃 Oxygen plants slider
* 📧 Newsletter subscription footer
* ♻️ Reusable React components

---

## 🛠️ Tech Stack

* React 18
* Vite
* Tailwind CSS
* Framer Motion
* React Icons

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/FloraVision.git
```

Navigate to the project folder:

```bash
cd FloraVision
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open your browser and visit:

```text
http://localhost:5173
```
